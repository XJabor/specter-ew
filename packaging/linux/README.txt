Specter EW desktop launcher
===========================

Keep the extracted files together, then run `bash install-desktop.sh`. Open
Specter EW from your desktop environment's application menu. The launcher
starts a local-only server and opens http://localhost:5000 in your browser.
Closing its terminal stops the server. Rerun the installer if you move the
extracted directory.

For LAN or custom-port operation, run SpecterEW directly from a terminal.
Optional DTED and GeoTIFF data may be placed in a local_data directory beside
the executable.
